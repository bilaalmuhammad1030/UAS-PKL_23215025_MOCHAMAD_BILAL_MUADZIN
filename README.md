## Judul Sistem

Sistem Administrasi dan Pengelolaan Soal Ujian Semester Berbasis Web di MA Mambaul Hikmah.

## Deskripsi Sistem

Sistem ini merupakan prototipe aplikasi berbasis web yang dibuat berdasarkan hasil observasi selama pelaksanaan Praktik Kerja Lapangan di MA Mambaul Hikmah.

Sistem bertujuan membantu proses pencatatan pengumpulan dan verifikasi soal ujian semester agar data dapat dikelola dengan lebih teratur.

## Latar Belakang

Selama pelaksanaan PKL, proses administrasi soal ujian semester membutuhkan pencatatan nama guru, mata pelajaran, kelas, semester, tanggal pengumpulan, dan status verifikasi soal.

Berdasarkan proses tersebut, dibuatlah prototipe sistem administrasi pengelolaan soal ujian semester berbasis web.

## Fitur Sistem

- Dashboard jumlah pengumpulan soal
- Form input pengumpulan soal
- Pencatatan nama guru
- Pencatatan mata pelajaran
- Pemilihan kelas dan semester
- Status verifikasi soal
- Pencarian data
- Hapus data
- Cetak laporan
- Penyimpanan data menggunakan Local Storage

## Teknologi yang Digunakan

- HTML
- CSS
- JavaScript
- Local Storage
- GitHub
- Vercel atau Netlify

## Cara Menjalankan Sistem

1. Unduh atau clone repository ini.
2. Buka folder proyek.
3. Klik dua kali file `index.html`.
4. Sistem akan terbuka melalui browser.
5. Sistem juga dapat dijalankan menggunakan ekstensi Live Server pada Visual Studio Code.
